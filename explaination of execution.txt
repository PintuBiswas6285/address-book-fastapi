Insructions and breif explaination to download configure and run the app form the public repo link.

Setup
1. Clone the repository
bash command
git clone https://github.com/your-username/address-book-fastapi.git

2. bash command
cd address-book-fastapi

3. Create Virtual Env
python -m venv myenv

4. Activate the Virtual Enviornement
venv\Scripts\activate #Windows source
venv/bin/activate # Linux/Mac

3. Install Dependencies
pip install -r requirements.txt

4. Run App
uvicorn app.main:app --reload

5. Open Browser
http://127.0.0.1:8000/docs

Docker Run
docker build -t address-api . docker run -p 8000:8000 address-api
